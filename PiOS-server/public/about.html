About PiOS Media

PiOS is a DIY media service based on The Ionic iOS Framework, RasperryPi, Node.js, PSQL, and AngularJS. PiOS is designed to allow users to easily stream media to iOS devices over a secure wireless LAN. Any user with the PiOS app can connect to our PiOS Media Servers. A PiOS Server admin can connect using a web browser to manage local media and user accounts.

Streaming Content

PiOS utilizes Ios streaming and UPnP protocols to cater to a wide variety of user devices. Our recomended UPnP media player is VLC. Accessing PiOS is as easy as connecting to EGGNOGG under the Local Network options.
